## 输出规则

1. **篇幅**: 3000-6000字（不含图表代码）
2. **格式**: Markdown，必须使用上方的完整结构
3. **表格**: 所有对比数据必须用表格呈现
4. **不确定性标注**: 
   - 推测性结论 → `⚠️ 推测`
   - 无法验证的数据 → `⚠️ 待验证`
   - 估算值 → `⚠️ 估算`
5. **数据纪律**: 所有关键数据标注来源和年份
